# HexaRemake

## Code Style

- Do NOT add comments of any kind to code (inline comments, block comments, XML doc comments/tooltips, summary tags).
- Do NOT use `#region` / `#endregion` blocks.
- Do NOT write `<summary>`, `<param>`, `<returns>`, or any other XML documentation tags.
- Names must be self-explanatory. If a name needs a comment to explain it, rename the identifier instead.
